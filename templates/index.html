About

This demonstration leverages a convolutional neural network trained on labeled images to perform binary classification: Non_Autistic vs Autistic. The model operates on resized RGB images (default 224Ã—224) and outputs class probabilities.

Method

The backend is built with Flask and TensorFlow/Keras. Uploaded images are preprocessed (standardized size and scaling) before inference. The application supports either a single sigmoid output or a two-unit softmax output.
